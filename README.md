A simple single-page CV built with semantic HTML.

This project is part of the Frontend Developer roadmap from roadmap.sh.

## Preview

![Project Preview](./preview.png)

## Features

* Semantic HTML structure
* SEO meta tags
* Open Graph tags
* SVG favicon
* Accessible markup
* Single-page layout

## Tech Stack

* HTML5
* CSS3
